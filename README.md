## Info.plist
- 記得要設定，在Finder / iPhone才能拉影片進去，單層資料夾支援

```xml
<key>UIFileSharingEnabled</key>
<true/>
<key>LSSupportsOpeningDocumentsInPlace</key>
<true/>
<key>UISupportsDocumentBrowser</key>
<true/>
```
